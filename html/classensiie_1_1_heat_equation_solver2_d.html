Detailed Description

Solves 2D heat equation with implicit finite differences.

Boundary conditions:

  • Neumann at x=0, y=0
  • Dirichlet at x=L, y=L

void ensiie::HeatEquationSolver2D::init_source ( double f)
private

Initialize heat source F(x,y).

Parameters
fHeat source amplitude (Celsius)

Four heat sources at symmetric positions: F(x,y) = tmax * f**2 on [L/6, 2L/6] x [L/6, 2L/6] F(x,y) = tmax * f**2 on [4L/6, 5L/6] x [L/6, 2L/6] F(x,y) = tmax * f**2 on [L/6, 2L/6] x [4L/6, 5L/6] F(x,y) = tmax * f**2 on [4L/6, 5L/6] x [4L/6, 5L/6] F(x,y) = 0 otherwise
